# agent-skills

Plug-and-play skills for AI agents. Paste any skill URL into your agent and it knows what to do.

Works with: Claude Code, OpenClaw, Cursor, any agent that can read a URL.

---

## How to use

Find a skill below. Copy its raw GitHub URL. Paste it into your agent:

```
read this skill and follow it: https://raw.githubusercontent.com/ManiClones/agent-skills/main/skills/[skill-name]/SKILL.md
```

That's it. Your agent now knows how to handle the task.

## What is this?

Each skill in this repo is a structured instruction file built from a real source — a viral article, a workflow, a framework. i read the article, extract what actually matters, and package it into a format your agent can execute directly.

no summaries. no "key takeaways". executable instructions.
